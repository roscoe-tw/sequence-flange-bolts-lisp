* sequence-flange-bolts-lisp
  這個程式是用來繪製法蘭螺栓鎖固順序的 autolisp 程式。

  載入 "sequence-flange-bolts-lisp.lsp" 後，在修改程式後，可以用 "RRR" 命令重新載入該程式。
  使用命令 "screw-sequence" 指定一個定點，就可以繪製法蘭螺栓鎖固順序圖。
